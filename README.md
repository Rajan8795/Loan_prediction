# Loan_prediction
Loan Approval Prediction System Using Machine Learning and Flask

📌 Project Overview

The Loan Approval Prediction System is a Machine Learning powered web application that predicts whether a loan application is likely to be approved or rejected based on applicant information.

The project combines Machine Learning, Flask, SQLite, and a modern responsive user interface to provide a complete end-to-end solution for preliminary loan eligibility assessment.

✨ Features

✅ User Registration and Login System

✅ Secure Password Hashing

✅ Session-Based Authentication

✅ Loan Approval Prediction

✅ Machine Learning Model Integration

✅ SQLite Database Management

✅ Responsive and Interactive Dashboard

✅ Real-Time Prediction Results

✅ Modern Cyber-Themed User Interface
